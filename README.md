#   📝 Projeto E-commerce
## 🖊 Descrição:
Desenvolvimento de um site e-commerce
<br/>
## 🛠 Tools and Languages:
- TypeScript
- MongoDB
- Node.js
- Mongoose
- Eslint
- Jest (teste unitário)

## Como usar
- Baixar dependências : entrar na pasta server
```
npm install
```
- Testar na API (Postman, Insomnia, outros):
```
npm run dev
```
- Testar com Jest 
```
npm run test
```
